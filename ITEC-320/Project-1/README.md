### Project #1

* * *

**How To Run:**
* Commandline Guide
    * For Non-Verbose Output
```
    gcc -c hiring.adb
    gnatmake hiring.adb
    hiring < hiring_data.txt
```
```
    * For Verbose Output
```    
```
    gcc -c hiring.adb
    gnatmake hiring.adb
    hiring -v < hiring_data.txt
```

* Unix Guide
    * For Non-Verbose Output
```
    $ gcc -c hiring.adb
    $ gnatmake hiring.adb
    $ ./hiring < hiring_data.txt
```

    * For Verbose Output
    
```
    $ gcc -c hiring.adb
    $ gnatmake hiring.adb
    $ ./hiring -v < hiring_data.txt
```
